# 🧠 NLP Mini App
A simple NLP web app built with Python, TextBlob, and Streamlit.

### Features
- Sentence correction using TextBlob
- Sentiment analysis (Positive / Negative / Neutral)
- Simple web app using Streamlit
- Button to control analysis

## How to run
pip install streamlit textblob
streamlit run app.py

### What I learned
- Basic NLP pipeline
- Using Python libraries (TextBlob)
- Building a simple web app with Streamlit
- Handling user input and UI
